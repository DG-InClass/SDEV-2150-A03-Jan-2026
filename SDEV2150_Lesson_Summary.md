# SDEV2150 - Weekly Lesson Summary

| Lesson | Title                                                       | Primary Focus Area        | Brief Description                                                                                       |
| :----: | :---------------------------------------------------------- | :------------------------ | :------------------------------------------------------------------------------------------------------ |
|   01   | JavaScript Review                                           | Review & Foundations      | Refreshes core JavaScript syntax, logic, and DOM interaction to ensure readiness for advanced concepts. |
|   02   | Web Components and Composition                              | Component Architecture    | Reviews the Web Component API and explores encapsulation and reusability through vanilla JS.            |
|   03   | Reactive Patterns and Advanced UI Communication             | Event-Driven Architecture | Introduces reactive programming, Observer and Pub/Sub patterns, and event-based UI updates.             |
|   04   | Advanced UI Communication with Custom Events                | Custom Event Handling     | Applies custom events to decouple component communication and improve modularity.                       |
|   05   | Asynchronous Programming in JavaScript                      | Async/Await & Promises    | Teaches event loop behavior, asynchronous control flow, and error handling for non-blocking code.       |
|   06   | Exercises: Asynchronous Programming                         | Practice Session          | Hands-on practice applying Promises, async/await, and async error-handling patterns.                    |
|   07   | Framework Foundations and Component Building (React + Vite) | Framework Setup           | Introduces React and Vite project setup, JSX syntax, and first functional components.                   |
|   08   | Building Components in React                                | Component Development     | Builds structured, reusable React components using props, children, and internal state.                 |
|   09   | Component Composition in React                              | Component Architecture    | Demonstrates component composition, container vs. presentational components, and reuse patterns.        |
|   10   | Styling Components in React                                 | Styling & Design Systems  | Applies Tailwind CSS utilities for component styling and responsive design.                             |
|   11   | UI Framework Integration with DaisyUI                       | Design Frameworks         | Integrates DaisyUI with Tailwind for rapid prototyping and theme-based UI consistency.                  |
|   12   | Event Handling and State in React                           | Interactivity & State     | Introduces event-driven UI and the `useState` hook to handle user input and dynamic updates.            |
|   13   | Lifting State and Component Communication                   | State Sharing             | Teaches lifting state up, prop drilling, and coordinated component communication.                       |
|   14   | Exercises: State and Communication                          | Practice Session          | Reinforces multi-component state and communication through hands-on challenges.                         |
|   15   | State-Driven UI in React                                    | Dynamic Rendering         | Demonstrates conditional and list rendering, derived state, and declarative UI updates.                 |
|   16   | Component Lifecycle in React                                | useEffect & Lifecycle     | Explores mounting, updating, and cleanup phases using `useEffect` for side effects.                     |
|   17   | Static Routing with React Router 7                          | Routing Fundamentals      | Introduces React Router v7 static routes, layout routes, and navigation with `<Link>`.                  |
|   18   | Dynamic Routing with React Router 7                         | Data-Driven Routing       | Covers dynamic URL parameters, `useParams()`, and nested routes for structured content.                 |
|   19   | Data Fetching Strategies in React                           | API Integration           | Introduces async data fetching with `fetch()` and `useEffect`, handling loading and errors.             |
|   20   | Exercises: Routing and Data Fetching                        | Practice Session          | Integrates routing and data-fetching skills in small project-based exercises.                           |
|   21   | Client Rendering Strategies in React                        | Rendering Models          | Examines client-side rendering, virtual DOM, and performance optimization.                              |
|   22   | Server Rendering Strategies in React                        | SSR Concepts              | Compares client vs. server rendering and explores SSR advantages and trade-offs.                        |
|   23   | Client Application State in React                           | Local & Global State      | Explores local and shared state patterns with `useState` and Context API.                               |
|   24   | Server-Side State in React with TanStack Query              | Server State Management   | Teaches fetching, caching, and synchronization of server data with TanStack Query.                      |
|   25   | State Synchronization and Testing with TanStack Query       | Advanced Data State       | Introduces invalidation, optimistic updates, and testing async components.                              |
|   26   | Exercises: Server State and Synchronization                 | Practice Session          | Reinforces mutation and caching strategies through guided exercises.                                    |
|   27   | Unit Testing React Applications                             | Unit Testing              | Teaches isolated component testing using Vitest and React Testing Library.                              |
|   28   | Test-Driven Development in React                            | TDD Workflow              | Applies the red-green-refactor cycle to build components guided by tests.                               |
|   29   | End-to-End Testing in React Applications                    | E2E Testing               | Simulates user workflows and navigation using Playwright or Cypress.                                    |
|   30   | Deployment Strategies for React Applications                | Course Integration        | Culminating project where students integrate all course concepts in a final app build.                  |
