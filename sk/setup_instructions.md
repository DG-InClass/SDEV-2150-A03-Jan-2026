# SDEV2150 React + Vite + Tailwind Setup (JS Only)

**Baseline:**  React, Vite 7, Tailwind CSS 3, ESLint (flat config).  
**Why:** Stable ecosystem for teaching; later lessons can annotate React 19 / Tailwind 4 upgrades.

---

## 1) Prerequisites
- Node.js LTS (>= 18)
- npm (bundled with Node)
- VS Code with ESLint extension enabled
- ESLint used as formatter (no Prettier)

Workspace settings (example):
```json
{
  "editor.defaultFormatter": "dbaeumer.vscode-eslint",
  "editor.formatOnSave": true,
  "eslint.format.enable": true
}
```

---

## 2) Install dependencies

From the project root:
```bash
# App runtime
npm i react react-dom

# Dev tools
npm i -D vite @vitejs/plugin-react-swc           tailwindcss postcss autoprefixer           eslint-plugin-react eslint-plugin-react-hooks eslint-plugin-jsx-a11y
```

> Keep existing ESLint (v9 flat config) from SDEV1150. We’ll just add React plugins.

---

## 3) Vite config

**vite.config.js**
```js
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react-swc';

export default defineConfig({
  plugins: [react()],
});
```

**index.html**
```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>SDEV2150</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.jsx"></script>
  </body>
</html>
```

**src/main.jsx**
```jsx
import React from 'react';
import { createRoot } from 'react-dom/client';
import './index.css';
import App from './App.jsx';

const root = createRoot(document.getElementById('root'));
root.render(<App />);
```

**src/App.jsx**
```jsx
export default function App() {
  return (
    <main className="p-6 max-w-3xl mx-auto">
      <h1 className="text-2xl font-bold mb-4">Hello React + Tailwind</h1>
      <p className="mb-2">First component in our Vite scaffold.</p>
    </main>
  );
}
```

## 4) Tailwind setup (v3)

Initialize Tailwind:
```bash
npx tailwindcss init -p
```

**tailwind.config.js**
```js
/** @type {import('tailwindcss').Config} */
export default {
  content: ['./index.html', './src/**/*.{js,jsx}'],
  theme: { extend: {} },
  plugins: [],
};
```

**postcss.config.js** (auto-generated by `-p`):
```js
export default {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
};
```

**src/index.css**
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## 5) ESLint (flat) additions for React

In `eslint.config.mjs`, add the following to your existing config:
```js
import react from 'eslint-plugin-react';
import hooks from 'eslint-plugin-react-hooks';
import a11y from 'eslint-plugin-jsx-a11y';

export default [
  // ...your existing configs
  {
    files: ['**/*.{js,jsx}'],
    languageOptions: {
      globals: { /* include browser globals */, JSX: true },
    },
    plugins: {
      react,
      'react-hooks': hooks,
      'jsx-a11y': a11y,
    },
    settings: { react: { version: 'detect' } },
    rules: {
      'react/react-in-jsx-scope': 'off',
      'react/jsx-uses-react': 'off',
      'react/jsx-uses-vars': 'warn',
      'react-hooks/rules-of-hooks': 'error',
      'react-hooks/exhaustive-deps': 'warn',
    },
  },
];
```

## 7) Notes for lesson authors

- **Scope for Lesson 05:** scaffold + JSX + props + basic composition. No state, no routing.
- **Styling:** Tailwind utilities only. DaisyUI comes later.
- **Linting:** ESLint as formatter; no Prettier required.
- **Starters:** Provide a zip with `/src` and config files above for students to start quickly.

---

## 8) Upgrade pointers (React 19 / Tailwind 4)

- **React 19:** New features like Actions (`useActionState`), `useOptimistic`, and `ref` as a normal prop. Great for forms and async UX. Not required for SDEV2150 basics—consider a bonus demo later.
- **Tailwind 4:** Moves config into CSS (e.g., `@theme`) and simplifies tooling. Migration from v3 can be shown as an advanced exercise; keep v3 during core instruction to avoid churn.

---

## 9) Troubleshooting checklist

- Blank page? Confirm `#root` exists in `index.html` and `src/main.jsx` uses `createRoot` correctly.
- Styles not applying? Check `tailwind.config.js` `content` globs and that `index.css` is imported in `main.jsx`.
- ESLint not formatting? Verify VS Code settings and the ESLint extension is enabled.
- Hot reload issues? Ensure only one dev server is running and that files are inside the Vite root.
